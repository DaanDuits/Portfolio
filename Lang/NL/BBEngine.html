Bare Bones - Game Engine

De Bare Bones engine is mijn eigen C++ game engine waaraan ik in mijn vrije tijd aan werk. Ik volg hiervoor de game engine series van the Cherno op youtube. Hij legd in zijn videos duidelijk uit hoe een game engine werkt en hoe je er een kan maken. uiteindelijk wil ik alle bestaande systemen (zoals het input systeem) zelf proberen te maken als ik klaar ben met de playlist volgen. Ik hoop hiervan veel over C++ te leren en het maken van games van een meer backend perspectief.

Ik post een nieuwe blog post elke Zondag.
