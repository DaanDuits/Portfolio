Opzetten van een game engine

Dit is de Application class deze class deffineerd alle functies die het programma nodig heeft om te werken.
// Application class namespace BareBones { class BB_API Application { public: Application(); virtual ~Application(); void Run(); }; // To be defined in a client Application* CreateApplication(); }
In de EntryPoint.h word de main functie gecalled waarin de door de client aangegeven app word geinstantieerd en zijn Run functie word aangeroepen.
#pragma once #ifdef BB_PLATFORM_WINDOWS extern BareBones::Application* BareBones::CreateApplication(); int main(int argc, char** argv) { auto app = BareBones::CreateApplication(); app->Run(); delete app; } #endif
In het Core.h script word er gecontrolleerd wat de BB_API is dit is nodig om de benodigde classes en functies makkelijker te importeren en te exporteren vanaf de engine's .dll file naar de app's exe.
#pragma once #ifdef BB_PLATFORM_WINDOWS #ifdef BB_BUILD_DLL #define BB_API __declspec(dllexport) #else #define BB_API __declspec(dllimport) #endif #else #error Bare Bones only supports Windows! #endif
